# MindMate - Atom Engine 4.0

Sistema de produtividade pessoal baseado na especificação técnica Atom Engine 4.0.

**Status:** 🟡 Alpha (Core Engine + Debug Console)

---

## 🚀 Quick Start

1. Acesse o projeto no Lovable
2. Crie uma conta ou faça login
3. Pressione `Ctrl+Shift+E` para abrir o Debug Console
4. Teste o ParsingEngine na aba "Input Test"

## 🛠️ Stack Tecnológica

- **Frontend:** React + Vite + TypeScript
- **Styling:** Tailwind CSS
- **Backend:** Lovable Cloud (Supabase)
- **State:** React Query + Zustand
- **Icons:** Lucide React
- **Dates:** date-fns
